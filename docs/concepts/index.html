    人工智能概念 #

    本节描述Spring AI使用的核心概念。我们建议仔细阅读它，以了解Spring AI是如何实现的。

    型号 #

    人工智能模型是设计用于处理和生成信息的算法，通常模仿人类的认知功能。

    提示 #

    提示是基于语言的输入的基础，指导人工智能模型产生特定的输出。

    Tell me a {adjective} joke about {content}.
    • 最近的研究论文
    • StringTemplate（字符串模板）

    提示模板 #

    创建有效的提示涉及建立请求的上下文，并用特定于用户输入的值替换请求的部分。

    Tell me a {adjective} joke about {content}.
    • StringTemplate（字符串模板）

    预埋件 #

    嵌入是文本、图像或视频的数字表示，用于捕获输入之间的关系。

    代币 #

    代币是人工智能模型工作方式的构建块。

    结构化输出 #

    AI模型的输出通常以java.lang.String的形式到达，即使您要求以JSON格式返回。

    • 结构化输出转换

    检索增强生成 #

    一种称为检索增强生成（RAG）的技术已经出现，以解决将相关数据合并到准确人工智能模型响应提示中的挑战。

    • ETL管道
    • ChatClient-RAG（聊天客户端-RAG）

    刀具调用 #

    大型语言模型（LLM）在训练后被冻结，导致知识过时，并且它们无法访问或修改外部数据。

    • 刀具调用

    响应用户请求有效评估人工智能系统的输出对于确保最终应用程序的准确性和有用性非常重要。
